# 202505310

$T$が小さいケースだと、1つのターゲットを完成させるのに使える平均ターン数が4ターンしかない。

- 2つの絵の具を出して2gにする
- 1gを画伯に差し出す
- 残りの1gを破棄する

といった一連の操作が丁度4ターンで出来るのでとりあえずこれを試す。

多くのケースでスコアは改善したが、絵の具を1g取り出すのに必要なコスト$D$の制約が厳しいケースでは悪化した。

- コスト$D$が大きい -> 絵の具を無駄遣いできない
- 最大ターン数$T$が小さい -> 調合に時間をかけられない(誤差を小さくすることが難しい)
